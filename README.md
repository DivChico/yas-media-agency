# @DivChico/zone-mui-template

A premium Next.js & TypeScript template library for rapid project development, featuring a comprehensive MUI Kit and pre-configured design system.

## 🚀 Features

- **Next.js 13+** - Optimized for performance and SEO.
- **TypeScript** - Full type safety across the application.
- **MUI (Material UI) v5** - A rich set of customizable UI components.
- **Framer Motion** - Smooth, production-ready animations.
- **Form Handling** - Integrated with `react-hook-form` and `yup`.
- **Pre-configured ESLint & Prettier** - Consistent code style out of the box.
- **Modern Design System** - Includes custom overrides for typography, palette, and shadows.

## 🛠️ Getting Started

### 1. Installation

This template is optimized for **NPM**. Ensure you have [Node.js](https://nodejs.org/) installed.

```bash
npm install
```

### 2. Development

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:8002`.

### 3. Building for Production

To create an optimized production build:

```bash
npm run build
```

To preview the production build:

```bash
npm run start
```

## 📂 Project Structure

- `src/components`: Reusable UI components.
- `src/theme`: Custom MUI theme configuration and overrides.
- `src/utils`: Helper functions and utilities.
- `src/types`: TypeScript definitions.
- `public`: Static assets (images, fonts, etc.).

## 📝 Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the application for production.
- `npm run lint`: Checks for linting errors.
- `npm run prettier`: Formats code using Prettier.
- `npm run clear-all`: Removes `.next` and `node_modules`.

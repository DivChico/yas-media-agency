# YAS Media Agency — Pitch Demo

> وصف المشروع: موقع تعريفي متكامل لوكالة تسويق رقمي تعمل بالذكاء الاصطناعي

---

## 📋 Description / وصف المشروع

**English:**
A professional single-page promotional website for **YAS Media Agency**, an AI-Native digital marketing agency based in Riyadh, Saudi Arabia. The site is built as a pitch demo to showcase the agency's capabilities, services, and brand identity to potential clients across KSA and the GCC. It features a modern dark-theme design with glass-morphism effects, bilingual support (Arabic/English), and full responsiveness across all devices.

**العربية:**
موقع إعلاني مكون من صفحة واحدة لـ **وكالة ياس ميديا**، وهي وكالة تسويق رقمي تعمل بالذكاء الاصطناعي ومقرها الرياض، المملكة العربية السعودية. تم بناء الموقع كعرض تقديمي لإظهار قدرات الوكالة وخدماتها وهويتها البصرية للعملاء المحتملين في المملكة والخليج. يتميز بتصميم عصري داكن مع تأثيرات الزجاج، ودعم ثنائي اللغة (عربي/إنجليزي)، واستجابة كاملة لجميع الأجهزة.

## 🌐 Bilingual Support / دعم اللغتين

- **English (LTR):** Default language
- **Arabic (RTL):** Toggle via header button
- Content, labels, placeholders, SEO meta tags all switch between languages
- Layout direction switches between LTR and RTL using `stylis-plugin-rtl`
